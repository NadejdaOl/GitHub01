1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

-на компьютере должен быть установлен git;
-после установки Git — указать ваше имя и адрес электронной почты,  каждый коммит в Git содержит эту информацию, и она включена в коммиты

git config --global user.name "Nadiia olkhovska"  
git config --global user.email nadusharikol@gmail.com

-можно выбрать текстовый редактор,который будет использоваться, если будет нужно набрать сообщение в Git( по умолчанию vim) 

git config --global ...

-сгенерировать и загрузить в гит ssh ключи
-созданный репозиторий клонируем

git clone git://...путь/до/скрипта (ссылка из гита  SSH)     (клонирует репозитарий в новую директорию)


2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

заходим в папку проекта  и выполняем команды:
-git init (для создания нового репозитария, выполняется только раз для первоначальной настройки нового репозитария)
-git add . (добавляются все файлы)
-git commit -m "Initial commit"(первый коммит)
-git remote add origin git@....
-git status(проверить все ли изменения запушены на гитхаб)
-git push -u origin master (запушиваем изменения в гит)


Как получить на локальный компьютер изменения с github.com

-git pull origin master (скачиваются новые коммиты)

Какую информацию нам дают команда :git status - используется для просмотра статуса файлов в вашем локальном репозитарии(отслеживаются, не отслеживаются файлы, изменены ли они)
git log - история коммитов,выводит имя автора и адрес электронной почты каждого коммита,дату
git remote -v - благодаря этой команде мы будем знать, что репозиторий связан с гит-хабом, для просмотра подключенных удаленных репозитариев
